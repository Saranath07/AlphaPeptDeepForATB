peptdeep.pipeline_api
===============================

High-level APIs for:

* transfer learing (:meth:`peptdeep.pipeline_api.transfer_learn`)
* library prediction (:meth:`peptdeep.pipeline_api.generate_library`)
* rescoring (:meth:`peptdeep.pipeline_api.rescore`)
* more will be added

All the default parameters of these functionalities
are controlled by the :data:`peptdeep.settings.global_settings`.
